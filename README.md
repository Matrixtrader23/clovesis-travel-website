# Clovesis Destinations – Modern Travel & Tourism Website

Clovesis Destinations is a visually stunning, high-performance tourism website frontend built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**. It features a modern premium blue aesthetic, smooth animations, and a fully responsive design tailored for a professional travel agency experience.

## ✨ Features

- **Premium Blue Aesthetic**: A custom-designed light blue theme with deep ocean blue accents.
- **Modern Sticky Navbar**: Transparent on hero, transitions to a frosted glass dark blue on scroll.
- **Interactive Hero Section**: Engaging entrance animations with high-quality visual backgrounds.
- **Dynamic Content Sections**:
  - Featured Destinations Grid.
  - Tour Packages with real-time pricing formatting.
  - Client Testimonials with a smooth Swiper.js carousel.
- **Responsive Detailed Pages**:
  - Destination listing page.
  - Tour Packages page.
  - Dynamic Package Details with an image gallery slider.
  - Professional Contact page with integrated Google Maps and contact form.
- **LKR Currency Integration**: All pricing is set in Sri Lankan Rupees (Rs.) with professional formatting.
- **Local Asset Management**: Fully configured to serve images locally from the `public` directory.

## 🚀 Technologies Used

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Sliders/Carousels**: [Swiper.js](https://swiperjs.com/)
- **Typography**: Google Fonts (Poppins & Inter)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/travel-website.git
   cd travel-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Visit the site:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `/src/app`: Main pages and routing logic.
- `/src/components`: Reusable UI components (Navbar, Footer, Cards, Sections).
- `/src/data`: JSON files for destinations and tour packages data.
- `/public/images`: Local image assets for the website.
- `tailwind.config.ts`: Custom color palette and typography configuration.
